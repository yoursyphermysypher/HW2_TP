# Помощник для приготовления любого блюда на всю семью

Проект для домашнего задания по ООП, тестированию и Git.

В проекте реализована система для работы с рецептами:
- `Ingredient` хранит название ингредиента, количество и единицу измерения;
- `Recipe` хранит рецепт, добавляет ингредиенты и масштабирует порции;
- `ShoppingList` формирует список покупок по одному или нескольким рецептам.

Основной ноутбук с заданием: `HW_2_OOP_Testing_Git.ipynb`.

## Установка

Склонируйте репозиторий и перейдите в папку проекта:

```bash
git clone https://github.com/yoursyphermysypher/HW2_TP.git
cd HW2_TP
```

Создайте и активируйте виртуальное окружение:

```bash
python -m venv .venv
source .venv/bin/activate
```

Установите зависимости:

```bash
pip install -r requirements.txt
```

## Воспроизводимость

После установки проект должен запускаться без ручных правок:

```bash
pytest
```

## Использование

Откройте ноутбук:

```bash
jupyter notebook HW_2_OOP_Testing_Git.ipynb
```

Запустите ячейки по очереди. Тесты в ноутбуке находятся в разделе `Тестирование` и запускаются через `ipytest.run()`.
