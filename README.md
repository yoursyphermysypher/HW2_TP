# HW2: OOP, Testing, Git

Проект для домашнего задания по технологиям программирования.

В ноутбуке `HW_2_OOP_Testing_Git.ipynb` реализована система для работы с рецептами:
- `Ingredient` хранит название ингредиента, количество и единицу измерения;
- `Recipe` хранит рецепт и позволяет добавлять ингредиенты, масштабировать порции и считать уникальные ингредиенты;
- `ShoppingList` формирует список покупок по одному или нескольким рецептам.

## Установка

Склонируйте репозиторий и перейдите в папку проекта:

```bash
git clone <ваш-репозиторий>
cd <папка>
```

Установите зависимости:

```bash
pip install -r requirements.txt
```

Запустите тесты:

```bash
pytest
```

## Использование

Откройте ноутбук:

```bash
jupyter notebook HW_2_OOP_Testing_Git.ipynb
```

Запустите ячейки сверху вниз. Тесты находятся в разделе `Тестирование` и запускаются через `ipytest.run()`.
